# 📱 Aplicativo Flutter  
**Aula de Dispositivos Móveis — Fatec Matão**

Este projeto foi desenvolvido para as aulas de Dispositivos Móveis utilizando o framework Flutter. O objetivo é auxiliar no aprendizado de desenvolvimento mobile com Dart e Flutter.

---

## 🚀 Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [VS Code](https://code.visualstudio.com/)
- [Android Studio](https://developer.android.com/studio)

---

## 🧩 Extensões recomendadas (VS Code)

Instale as seguintes extensões:

- Flutter  
- Dart  
- Flutter Widget Snippets  

---

## ⚙️ Configuração do ambiente

### 1. Configurar o Android Studio

1. Abra o Android Studio  
2. Clique em **More Actions**  
3. Acesse **Virtual Device Manager**  
4. Crie um dispositivo virtual (emulador) de sua preferência  

---

### 2. Criar o projeto Flutter no VS Code

1. Abra o VS Code  
2. Pressione **Ctrl + Shift + P**  
3. Digite `Flutter` e selecione **Flutter: New Project**  
4. Escolha **Application**  
5. Selecione a pasta onde o projeto será salvo  
6. Defina o nome da aplicação  

---

## ▶️ Executando o projeto

1. Inicie o emulador pelo Android Studio ou conecte um dispositivo físico  
2. No VS Code, abra o projeto  
3. Execute o comando:

```bash
flutter run
```
